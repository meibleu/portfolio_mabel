Teamfight Tactics Client

After analyzing the existing client, I identified several key usability issues that drove the direction of this redesign:


Storage Efficiency: The requirement to install the entire League of Legends game ecosystem creates unnecessary storage bloat for users who exclusively play Teamfight Tactics.

Asset Visualization: There is no functionality to view high-resolution or enlarged details of cosmetic assets.

Lobby Layout: The current split-layout places players on opposite sides of the screen. This visual hierarchy increases cognitive load, making it difficult to track player status at a glance.

Key Design Decisions:


Standalone Application: Developed a dedicated TFT environment by stripping away non-essential League of Legends elements to reduce cognitive load.

Social Consistency: Preserved the legacy Friend List layout to maintain user familiarity and ease of access.

Enhanced Navigation: Relocated primary tabs to a vertical left-hand sidebar with increased font weight, adopting a modern UI standard similar to the Valorant home screen.

Mobile Parity: Mirrored the mobile clientâ€™s layout for the Battlepass and "Objectives" to create a unified experience across platforms.

Engagement Optimization: Utilized available viewport space to showcase "Treasure Realm" cosmetics, increasing visibility for in-game content.
